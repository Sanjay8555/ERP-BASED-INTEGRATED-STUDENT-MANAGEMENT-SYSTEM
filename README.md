# Integrated Student Management System ERP

A modern ERP-style student management platform built with React, TypeScript, Vite, and a Spring Boot backend. The application is designed to help institutions manage student records, academic modules, services, and billing-related workflows in a unified interface.

## Features

- Student dashboard overview
- Student management module
- Academic module tracking
- Service and receipt management
- Responsive UI for administrative use
- Backend API structure for student operations

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- CSS

### Backend
- Java
- Spring Boot
- REST API structure

## Project Structure

- src/ - frontend source files
- backend-springboot/ - Java backend services
- database/ - SQL schema files
- docs/ - project documentation

## Getting Started

### Prerequisites
- Node.js and npm
- Java JDK 17 or higher
- Maven or your preferred Java build tool

### Frontend Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

### Backend Setup

1. Navigate to the backend folder:
   ```bash
   cd backend-springboot
   ```
2. Build and run the Spring Boot application:
   ```bash
   ./mvnw spring-boot:run
   ```
   If Maven Wrapper is not available, use:
   ```bash
   mvn spring-boot:run
   ```
